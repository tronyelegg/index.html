# NIKKE 스펙 카드 제너레이터 — 간편 사용 가이드

> 블라블라링크(시프티패드)의 도감 데이터를 바탕으로 깔끔한 파티 스펙 카드를 자동 생성합니다.
> **PC 및 모바일 웹 브라우저 지원** (PC Chrome 환경 권장).

> 💡 **별도의 앱 설치나 확장 프로그램이 전혀 필요 없는 순수 웹 도구입니다.**
> 
> ✅ 데이터 추출 과정은 카드를 만들 때마다 **원할 때만 실행**하시면 됩니다.

---

## Step 1. 시프티패드 접속 및 로그인

본인의 니케 데이터를 불러오기 위해 공식 웹사이트에 접속해야 합니다.

1. 브라우저에서 **블라블라링크 도감 페이지**에 접속합니다.
2. 본인의 레벨 인피니트(Level Infinite) 계정으로 로그인합니다.
3. 도감 화면에 내 니케 목록이 정상적으로 표시되는지 확인합니다.

---

## Step 2. 데이터 추출 (북마크릿 실행)

> ⏭️ **이미 .json 추출 파일을 가지고 있다면 이 단계를 건너뛰고 Step 4로 가세요.**

1. 스펙 빌더 사이트에서 **[북마크릿 복사]** 버튼을 눌러 스크립트 코드를 복사합니다.
2. 띄워두었던 블라블라링크 도감 페이지로 돌아옵니다.
3. **방법 A (PC 권장):** 개발자 도구(F12)를 열고 `Console(콘솔)` 탭의 입력창에 코드를 붙여넣고 엔터를 칩니다.
4. **방법 B (모바일/주소창):** 브라우저 주소창의 기존 주소를 다 지우고 `javascript:`를 직접 타이핑한 뒤, 그 바로 뒤에 복사한 코드를 붙여넣고 엔터를 칩니다.

---

## Step 3. 스펙 빌더로 자동 전송

1. 스크립트가 실행되면 도감 화면에 데이터 추출 완료 팝업 창이 뜹니다.
2. **[🚀 1. 스펙 빌더로 자동 전송]** 버튼을 클릭합니다.
3. 새 창으로 스펙 빌더 사이트가 열리며 데이터가 즉시 로드됩니다.

> ⚠️ **주의사항:** 브라우저에 **'팝업 차단'**이 설정되어 있으면 새 창이 열리지 않습니다. 이때는 팝업 차단을 해제하시거나, 팝업 창의 '2번 버튼'을 눌러 JSON 파일을 다운로드하여 수동으로 진행하세요.

---

## Step 4. 데이터 직접 업로드 (선택)

> ⏭️ **Step 3에서 자동 전송을 완료했다면 건너뛰세요.**
>
> 🆕 다운로드한 JSON 파일이 있다면 언제든지 이 방법으로 불러올 수 있습니다.

1. [스펙 빌더 페이지]에 접속합니다.
2. **[파일 선택]** 버튼을 누르고 추출했던 `nikke_spec_...json` 파일을 업로드합니다.
3. `✅ 데이터 로드 완료!` 메시지가 뜨면 하단의 **[스펙 카드 생성]** 버튼을 누릅니다.

---

## Step 5. 스펙 카드 꾸미기 및 저장

데이터가 로드된 빌더 화면에서 자유롭게 파티를 구성할 수 있습니다.

1. 화면 하단의 **'내 니케 목록'**에서 원하는 니케를 클릭하여 상단 파티 슬롯에 편성합니다. (최대 5명)
2. 상단 옵션 바에서 **전투력 연산 기준** (싱크로 현재 레벨 / Lv.40 유레 기준 / Lv.400 솔레 기준)을 자유롭게 선택합니다.
3. 다국어(KR/EN/JP)를 변경하거나 오버로드 옵션 합산 노출 여부를 조작해 봅니다.
4. 모든 세팅이 끝나면 우측 상단의 **[📸 스펙 카드 이미지 저장]** 버튼을 눌러 고화질 이미지를 다운로드합니다.

---

## ⚠️ 꼭 알아둘 것

- **미획득 니케는 상세 데이터를 알 수 없습니다.** 시프티패드 구조상 명함(LB0)조차 없는 미보유 니케는 스킬이나 오버로드 장비 정보를 불러오지 못합니다.
- **전투력과 옵션 수치에 미세한 오차가 있을 수 있습니다.** 오버로드 옵션 합산 수치나 최종 전투력은 인게임 내부의 보이지 않는 소수점 반올림 처리 방식으로 인해 0.1~0.2% 또는 전투력 수십 정도의 오차가 발생할 수 있습니다.
- **데이터는 절대 수집되지 않습니다.** 이 사이트는 개인의 게임 데이터를 서버로 전송하지 않습니다. 모든 처리는 사용자 본인의 브라우저 내에서만 이루어지며, 탭을 닫으면 데이터는 완전히 증발합니다.

---

## 🔧 안 될 때

| 증상 | 해결 |
|------|------|
| [자동 전송] 버튼을 눌러도 새 창이 열리지 않음 | 브라우저 주소창 우측이나 설정에서 **'팝업 차단 해제'**를 하셔야 합니다. 혹은 JSON 파일 다운로드 기능을 이용해 수동으로 업로드하세요. |
| 주소창에 복붙했는데 구글 검색만 됨 | 주소창에 복붙할 때 보안상 맨 앞의 `javascript:` 글자가 자동으로 지워지기 때문입니다. 직접 `javascript:`를 타이핑한 후 뒤에 코드를 붙여넣으세요. |
| 신규 캐릭터 이미지가 안 보임 (엑스박스) | 신캐 출시 직후에는 툴 내부 DB나 시프티패드 CDN에 이미지가 아직 갱신되지 않았을 수 있습니다. 추후 자동 갱신됩니다. |
| 추출기가 아예 작동하지 않음 | 시프티패드 측의 API 구조가 변경되었을 가능성이 높습니다. 개발자(게시글)에게 제보해 주세요. |
| 모바일에서 이미지 저장이 안 됨 | 일부 모바일 내장 브라우저 환경에서는 이미지 캡처 권한이 제한될 수 있습니다. Chrome 앱이나 Safari 앱을 직접 켜서 사용해 주세요. |
